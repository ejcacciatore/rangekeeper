# KeyLedger — Real Estate Transaction Intelligence Platform

## What It Is
Mobile-first, cloud-native platform that modernizes residential real estate transactions by turning fragmented deal activity into a structured, timestamped, trusted transaction layer.

## Core Problems
1. Deal workflows fragmented across agents, buyers, sellers, lenders, attorneys, inspectors, documents
2. Agents waste time manually coordinating signatures, POF, inspection reports, communication
3. Transaction behavioral/workflow data is lost after closing — never becomes reusable intelligence
4. Buyers/agents need faster, safer, more trusted way to verify funds, lock intent, coordinate offers

## Platform Layers

### Buyer Verification Layer (SoftLock)
- Scan property QR code → verify funds privately → deposit binder via mobile escrow → verified status strengthens offer
- "SoftLock" / proof flow is central to buyer experience

### Agent Operating Layer (AgentAmp)
- Command center dashboard for realtors
- Manage active buyers, monitor SoftLocks, verify client fund status without raw bank data, track pending offers

### Immutable Transaction Timeline (HomeLedger)
- Structured, timestamped events: listing → showing → offer → inspection → mortgage → closing
- Documents (contracts, disclosures, inspections) attach to specific deal events

### AI Matchmaking & Discovery
- Private discovery between passive buyers and non-listed sellers
- AI compatibility scoring, anonymous until mutual opt-in

### Relationship Graph & Marketplace
- Completed transactions build trusted network: agents, clients, brokers, inspectors, attorneys, providers
- Supports provider rankings, referral intelligence, embedded trust

### Data Moat / Intelligence Layer
- Structured behavioral dataset from transaction events, timing, provider performance, process outcomes
- Predictive timelines, provider scoring, future automation models

## Positioning
- Mobile-first, cloud-native, trust-driven, verification-oriented
- Workflow intelligence, not just document storage
- Coordination layer + structured event capture
- Immutable transaction history
- AI-enhanced discovery and recommendations
- Future-ready for smart contracts and digital asset verification
- Agent = orchestrator (NOT being disintermediated)

## Business Model
- SaaS subscriptions (agents)
- Transaction fees (secure binder deposit / escrow flows)
- Marketplace / referral fees (connected service providers)
- Long-term: digital asset verification + advanced market/transaction intelligence

## Staged Roadmap
1. **MVP**: Workflow capture — agent accounts, transaction timelines, document uploads, client portals
2. **Intelligence Layer**: AI matchmaking, predictive analytics, automated provider recommendations
3. **Network Expansion**: Home inspection databases, digital title chain/transfer, possibly fractional ownership infrastructure

## Framing
KeyLedger is building the secure transaction memory and intelligence layer for residential real estate. It transforms messy, human-driven coordination into a structured ledger of events, documents, validations, and relationships. Helps agents close faster, buyers prove credibility earlier, and creates a long-term proprietary dataset around how deals actually move.
