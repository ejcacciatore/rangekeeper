### Key Learnings

**Subagent Pattern Works Well:**
- Spawning dedicated subagent for project-specific work = faster, focused, persistent
- Haiku 4.5 handles context loading + status reports for $0.01-0.03 vs. Sonnet at $0.15+
- Can track separate token budget per project
- Reduces main session context bloat

**Usage Monitor Essential:**
- Visibility into token costs prevents budget surprises
- Daily/weekly trends help optimize model selection
- Efficiency ratings guide when to escalate to Opus vs. stay on Sonnet/Haiku

**Next.js Vercel Pattern:**
- Zero-config deployment (GitHub → Vercel → Done)
- API routes handle backend logic without separate server
- Free tier sufficient for personal/small business use
- Can migrate to Postgres later without rewriting frontend
