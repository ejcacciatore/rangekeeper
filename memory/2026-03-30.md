## Phase 2 Feature Planning (14:02-14:07 UTC)

**User Vision (Massimo via Rico):**
- Click a class → opens class detail page with:
  - Assignments for that class (priority-sorted: urgent first)
  - Grades in that class (overall + category breakdown)
  - Messages for that class
  - Study tips (AI + teacher announcements + student notes)
  - Deep link to Blackboard to view directly
- Main dashboard keeps existing "Upcoming Assignments" but adds:
  - All assignments from all classes (chronological default)
  - Filters: Due Date, Status, Priority, Class (multi-select)
  - Grouping options (by class or by priority)
- Visual indicators for urgency:
  - Color priority bars (red/urgent, yellow/soon, green/later)
  - Gentle pulse animation for urgent items (ASD-optimized, not jarring blink)
  - Pulse triggers: Overdue OR Due Today OR Failed/Low Grade

**Design Choices Made (ASD-optimized):**
- Pulse animation (fade in/out, 2-3 sec cycle) not harsh blinking
- Full page navigation (not modal) for clarity
- Sensory-friendly: no overstimulation, high contrast, whitespace
- Toggle filters (not dropdowns) for accessibility
