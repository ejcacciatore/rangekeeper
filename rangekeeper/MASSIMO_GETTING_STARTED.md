## 🔔 Color Codes

- 🟢 **Green (NORMAL):** More than 1 day away
- 🔵 **Blue (UPCOMING):** 1-3 days away
- 🟠 **Orange (HIGH):** Due soon, less than 24 hours
- 🔴 **Red (URGENT):** Overdue or due in next 6 hours

## ❓ What's Missing?

The dashboard is showing your **Spring 2026 courses** from the sample data.

To get **real live data** from Blackboard:

### Step 1: Install the Chrome Extension
1. Go to `chrome://extensions/`
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select the extension folder
5. You'll see 🎯 icon in the toolbar

### Step 2: Go to Blackboard
1. Visit `https://ualearn.blackboard.com`
2. The extension will scrape your courses and assignments
3. Data syncs to the dashboard (if backend is running)

## 🐛 Troubleshooting

**Dashboard is empty?**
- Reload the page (F5)
- Make sure JavaScript is enabled
- Check browser console (F12) for errors

**Extension not working?**
- Make sure it's installed (chrome://extensions/)
- Reload the extension (refresh button)
- Open Blackboard and check console (F12 → Console)
- Look for `[RangeKeeper]` messages

**Data is old?**
- Dashboard updates once per day (3 AM UTC)
- For real-time, we're building a better version
